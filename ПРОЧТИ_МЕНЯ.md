# 🚀 Note Manager - Быстрый старт в RAD Studio 13

## ⚡ Минимальные шаги для запуска (5 минут)

### 1. Откройте проект в RAD Studio 13
```
File → Open Project → выберите /workspace/NoteManager.dproj
```

### 2. Скачайте SQLite3.dll
- Перейдите на https://www.sqlite.org/download.html
- Скачайте **sqlite-dll-win64-x64-*.zip** (для 64-bit) или **sqlite-dll-win32-x86-*.zip** (для 32-bit)
- Распакуйте `SQLite3.dll` в папку `/workspace/`

### 3. Настройте пути поиска
```
Project → Options → Delphi Compiler → Search Path
Добавьте: $(PROJECT)\src;$(PROJECT)\src\app;$(PROJECT)\src\domain;$(PROJECT)\src\application;$(PROJECT)\src\infrastructure;$(PROJECT)\src\presentation
```

### 4. Соберите и запустите
```
Ctrl+F9 (Build)
F9 (Run)
```

## 🎯 Структура приложения

```
Note Manager (Clean Architecture)
│
├── Domain Layer         - Бизнес-логика (не зависит от UI/БД)
```

## 🐛 Если что-то пошло не так

1. **Ошибка "Cannot find unit"** → Проверьте пути поиска (шаг 3)
2. **Ошибка "SQLite3.dll not found"** → Скопируйте DLL в папку проекта
3. **Ошибка компиляции** → Убедитесь, что все файлы `.dfm` добавлены в проект
4. **Access Violation** → Запустите от имени администратора
